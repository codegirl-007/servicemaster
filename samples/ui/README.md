# FSM UI ideation samples

Static HTML/CSS mockups for field-service management workflows. **Not connected to the application.**

## View locally

Open `index.html` in a browser, or serve the folder:

```sh
cd samples/ui && python3 -m http.server 8765
```

Then visit http://localhost:8765/

## Screens

| File | Audience | Purpose |
|------|----------|---------|
| `index.html` | All | Gallery and links to mockups |
| `office-dashboard.html` | Office / dispatcher | Morning ops snapshot |
| `dispatcher-board.html` | Dispatcher | Day schedule grid + unassigned queue |
| `technician-day.html` | Technician | Mobile day list |
| `job-detail.html` | Technician / office | Job detail, checklist, timeline |

Shared styles: `styles.css`
